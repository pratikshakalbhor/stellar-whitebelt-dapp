# Stellar Payment dApp – Level 1 (White Belt)

## Project Description

This project is created as part of the **Stellar Journey to Mastery – Level 1 (White Belt)** challenge.

It is a simple React-based Stellar dApp that connects to the Freighter wallet, shows the wallet balance, and allows sending XLM on the Stellar Testnet.
The goal of this project is to learn basic wallet connection, balance handling, and transaction flow in a beginner Web3 application.

---

## Features

* Connect Freighter Wallet
* Display connected wallet address
* Show XLM balance
* Send XLM on Stellar Testnet
* Show transaction success message

---

## Tech Stack

* React.js
* Freighter Wallet
* Stellar Testnet
* Stellar SDK
* CSS

---

## Setup Instructions (Run Locally)

1. Clone the repository:

```
git clone <your-repo-link>
```

2. Open the project folder:

```
cd your-project-folder-name
```

3. Install dependencies:

```
npm install
```

4. Start the application:

```
npm start
```

5. Open in browser:

```
http://localhost:3000
```

Make sure Freighter Wallet is installed and connected to Testnet.
